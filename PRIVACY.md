# Privacy Policy — SAM Table Filter

_Last updated: July 20, 2026_

SAM Table Filter is a browser extension that displays and filters the SAM Table
stock recommendations from members.trulyrichclub.com for users who are logged
into that site.

## Data collection

**This extension does not collect, transmit, or share any user data.**

- No analytics, telemetry, tracking, or advertising of any kind.
- No data is ever sent to the developer or to any third party.
- The extension makes no network requests to any server other than the page
  the user is already viewing on members.trulyrichclub.com.

## Local storage

The extension stores the following **only on your device**, using the
browser's extension storage:

- A cached copy of the SAM Table recommendations parsed from the page, so the
  popup can display them without re-reading the page every time.
- Your starred favorites.
- UI preferences (such as light/dark theme).

This data never leaves your browser. Uninstalling the extension deletes it.

## Permissions

The extension only runs on `members.trulyrichclub.com`. It reads the SAM Table
page using your own logged-in session, solely to display its contents in the
extension popup. It does not access any other website.
